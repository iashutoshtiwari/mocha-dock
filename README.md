> [!WARNING]
> **This project is in very early development.** Expect bugs, missing features, and breaking changes.

Mocha is a dock based on plasma frameworks that provides an elegant and intuitive experience for your tasks and plasmoids. It animates its contents by using a parabolic zoom effect and tries to be there only when it is needed.

This is a fork of Latte Dock, ported to KDE Plasma 6 and Wayland, and rebranded as Mocha Dock.

## Features
- **Wayland-only:** Optimized for the modern Linux display server.
- **Plasma 6:** Built with Qt6 and KDE Frameworks 6.
- **Rebranded:** A fresh identity as Mocha Dock.

## Installation

> [!NOTE]
> Currently, only **Arch Linux** is officially supported.

To install Mocha Dock on Arch Linux, you can clone the repository and build the package using `makepkg`:

```bash
git clone https://github.com/iashutoshtiwari/mocha-dock.git
cd mocha-dock
makepkg -si
```

## Run Mocha Dock
Mocha is ready to be used by executing:
```bash
mocha-dock
```
or by activating **Mocha Dock** from your applications menu.
